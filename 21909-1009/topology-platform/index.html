ปวช.1

จบกิจกรรมนี้ นักเรียนจะ สร้างผังเครือข่าย · ตั้งรหัส Wi-Fi กันคนแปลกหน้าเกาะ · ทดสอบเชื่อมต่อ-ส่งข้อมูล ได้เองบนมือถือ

🛠️

โหมดสร้างเอง (Builder)

วางอุปกรณ์ · เชื่อมสาย · จำลองส่งข้อมูล ออกแบบเครือข่ายอิสระตามใจ

 เริ่มเลย 🔒

ใบงานที่ 7 · ตั้ง Wi-Fi ให้ปลอดภัย

ใส่ชื่อ-เลขที่ → วาง Router/AP + คอม → ตั้งรหัส Wi-Fi กันคนแปลกหน้าเกาะ → เชื่อมคอมเข้า Wi-Fi ด้วยรหัส · เช็กภารกิจอัตโนมัติ

 เริ่มเลย
📱 วิธีใช้: แตะ "โหมดสร้างเอง" → เลือกอุปกรณ์จากพาเลต → แตะวางบนกระดาน → สลับโหมด "เชื่อม" ต่อสายระหว่างอุปกรณ์ → กด "จำลองส่งข้อมูล" ดูแพ็กเก็ตวิ่ง
เปิดได้ทั้งมือถือและคอม ไม่ต้องต่อเน็ต งานจะถูกเก็บไว้ในเครื่องอัตโนมัติ
